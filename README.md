# Academic QA Testing Project

## Project Description
This repository contains an academic QA testing project developed within the Quality Assurance Tester Academy.
The objective of this project was to design test cases based on a provided requirements specification and ensure requirement coverage through a Requirement Traceability Matrix.

## QA Activities
* Test case design based on system requirements
* Creation of a Requirement Traceability Matrix (RTM)
* Verification of requirement coverage

## Project Artifacts
This repository includes the following testing artifacts:
* **Test Cases** – A set of test cases designed to validate system functionality
* **Requirement Traceability Matrix** – Mapping between requirements and test cases to ensure full test coverage

## Tools Used
* Microsoft Excel
* Manual testing techniques
  
# Notes
The testing artifacts included in this repository were originally created in Portuguese as part of the Academy program.
